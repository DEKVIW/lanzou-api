API 接口文档

1. 接口概述

本API提供蓝奏云分享链接的解析服务，可获取直接下载地址和文件信息。支持带密码和无密码的链接。

接口地址 https://api.example.com/parse
请求方式 POST
数据格式 application/json
2. 请求参数
参数名 类型 必填 说明
url string 是 蓝奏云分享链接，支持所有蓝奏云域名
password string 否 分享链接的访问密码（如有）
3. 响应参数
参数名 类型 说明 示例值
code integer 状态码：200成功，400请求错误，500服务器错误 200
msg string 状态描述 "解析成功"
data.download_url string 文件直接下载地址 "https://xxx.lanzoui.com/file/xxx"
data.filename string 文件名称 "example.zip"
data.file_size string 文件大小 "1.5 MB"
data.upload_time string 上传时间(YYYY-MM-DD) "2024-01-20"
data.file_type string 文件类型 "压缩包"
data.uploader string 上传者名称 "用户名"
4. 请求示例
# cURL
curl -X POST "https://api.example.com/parse" \
     -H "Content-Type: application/json" \
     -d '{"url": "https://www.lanzoui.com/xxxxx", "password": "1234"}'

# Python
import requests

response = requests.post(
    "https://api.example.com/parse",
    json={
        "url": "https://www.lanzoui.com/xxxxx",
        "password": "1234"  # 可选
    }
)
result = response.json()
5. 响应示例
# 成功响应
{
    "code": 200,
    "msg": "解析成功",
    "data": {
        "download_url": "https://xxx.lanzoui.com/file/xxxxx",
        "filename": "example.zip",
        "file_size": "1.5 MB",
        "upload_time": "2024-01-20",
        "file_type": "压缩包",
        "uploader": "用户名"
    }
}

# 错误响应
{
    "code": 400,
    "msg": "无效的蓝奏云链接"
}
6. 错误代码说明
状态码 说明 处理建议
200 解析成功 成功获取下载链接和文件信息
400 请求参数错误
  • 检查链接格式是否正确
  • 检查链接是否可访问
  • 如提示需要密码，请添加password参数
429 请求过于频繁 请降低请求频率，建议间隔1秒以上
500 服务器内部错误
  • 服务器解析失败，请稍后重试
  • 如持续出现请联系管理员
常见错误信息说明：
错误信息 说明
"无效的蓝奏云链接" 链接格式错误或不是有效的蓝奏云分享链接
"该链接需要访问密码" 分享链接设置了密码，需要在请求时提供password参数
"密码错误" 提供的访问密码不正确，请检查后重试
"请求过于频繁" 超出频率限制，请降低请求频率
"解析失败" 服务器解析出错，可能是链接失效或暂时无法访问
注意事项：
  • 请控制请求频率，建议每秒不超过1次请求
  • 返回的直链地址有效期较短，请及时使用
  • 如遇到解析失败，可以稍后重试
  • 本接口仅供学习交流使用，请勿用于商业用途
使用建议：
  • 建议在请求头中添加 User-Agent 以提高成功率
  • 批量解析时建议加入适当延时（1秒以上）
  • 如需解析大量链接，建议分批进行
  • 遇到解析失败时，可以查看返回的错误信息进行排查
